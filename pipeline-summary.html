Repair Engine

Iterative improvement until
the design is professional.

The Repair Engine runs up to 8 iterative passes. If the minimum score of 0.78 is not reached, anchors regenerate with a new seed offset and the pipeline restarts from scratch.

01
Density Normalization
Redistributes stems to match zone field density targets. Overcrowded zones shed stems; sparse zones absorb them.
02
Cluster Cohesion
Pulls scattered stems toward their nearest anchor point, strengthening visual groupings and reinforcing the focal cluster.
03
Silence Arc Enforcement
Clears the designated silence arc of non-greenery stems, creating the intentional negative space that makes the focal cluster feel powerful.
04
Focal Clarity
Ensures focal blooms are unobstructed, correctly depth-layered, and meet minimum angular separation requirements.

Blueprint DNA

Every design is deterministic,
versioned, and reproducible.

Evercrafted is not a random generator. Identical inputs always produce identical designs. Every wreath receives a Blueprint DNA fingerprint — a cryptographic record of the design's structural identity.

Deterministic Seed Generation Rule
seed = hash(
  memory_input +
  inventory_signature +
  formula_id
)
  • Product identification — every blueprint has a unique, human-readable fingerprint for cataloging and search.
  • Blueprint resale — designs can be licensed and resold because they are fully reproducible from the seed alone.
  • Version control — design iterations are tracked by seed, enabling rollback and comparison across versions.

Engine Output Layer

One solved design.
Four deliverables.

The Blueprint & Render Engine serializes the complete solved placement set into four distinct output types — each serving a different user need, from hands-on building to photorealistic visualization.

.json
Blueprint JSON
Machine-readable design record containing every stem's polar coordinates, role, depth, and direction vector. Versionable, resellable, and fully reproducible from the seed.
Markdown / PDF
Builder Guide
Step-by-step placement instructions for the wreath maker. Sequenced from greenery skeleton through focal blooms, secondary, accents, and filler. Human-readable and printable.
.svg
SVG Preview
Visual canvas preview of the polar placement layout. Shows the full ring with stem positions, depth layers, and zone boundaries rendered as a scalable vector graphic.
Text Prompt
Render Prompt
Midjourney-ready photorealistic render prompt generated from the palette, formula, and genome data. Produces a photorealistic visualization of the finished wreath.
Builder Guide Sequence

Five steps from frame
to finished wreath.

The Builder Guide sequences placement instructions in depth-layer order — starting with the structural skeleton and ending with the focal statement. This ensures every layer is set before the next is placed.

1
Place Greenery Skeleton
Establish the structural base with gesture and greenery stems. This defines the wreath's silhouette, working radius, and overall density distribution.
2
Place Focal Blooms
Position the primary focal stems at their designated polar coordinates within the focal arc. These are the visual anchors of the entire design.
3
Place Secondary Blooms
Add secondary blooms to support and frame the focal cluster, building depth and visual rhythm around the primary anchor points.
4
Add Accents
Place accent stems to introduce texture contrast, color punctuation, and directional movement between the larger bloom masses.
5
Fill with Filler Stems
Complete the design with filler stems to close gaps, smooth transitions, and achieve the target density in each zone field.
